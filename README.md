# Entwood Finance Frameworks

This folder contains four original, practitioner-focused papers and frameworks developed for conviction-driven investing across different time horizons and styles.  

All documents are shared freely: make something useful, make it available, let those who need it find it. No paywalls, no gatekeeping — just clear, reproducible processes that anyone can adapt to their own thesis, risk temperament, and goals.

## The Four Frameworks

1. **Constructing a Conviction-Driven Long-Term Thematic IPS**  
   A step-by-step guide to building a rules-based, long-term thematic investment policy statement anchored in structural mega-trends. Includes thesis definition, core-satellite allocation, momentum/quality scoring, layered protection mechanisms, governance cadence, and optional autonomic evolution.  
   File: `Constructing a Conviction-Driven Long-Term Thematic IPS - A Step-by-Step Framework.md`

2. **Constructing a High-Yield IPS**  
   A framework for designing an income-focused investment policy statement that prioritizes stable yield, capital preservation, and consistent cash flow in a rules-based structure.  
   File: `Constructing a High-Yield IPS.md` (or exact filename you used)

3. **Constructing a BNF (BTF-Style Daytrading) IPS**  
   A disciplined, short-term trading IPS inspired by Japanese day-trading principles — fast execution, strict risk controls, and high-conviction setups for intraday/intra-week horizons.  
   File: `Constructing a BNF IPS.md` (or exact filename you used)

4. **Constructing Meta-Kaizen: A Self-Evolving Governance Loop**  
   Introduces Meta-Kaizen — a lightweight, autonomic Kaizen mechanism that lets any governance document (including IPS) continuously improve by drawing from orthogonal domains. Features the Kaizen Variation Score (KVS) with novelty rotation, impact weighting, complexity penalties, and smooth timeliness decay.  
   File: `Meta Kaizen self evolving algorithm.md` (or exact filename you used)
