## The Law — Stated Simply

A network is any system that transmits information
across time between distinguishable states.

From that definition two properties emerge as
necessary and sufficient:

**Structure** — the physical instantiation of
distinguishability. Differential association
between nodes.

**Rhythmicity** — the oscillatory carrier that
maintains signal against dissipation. The physical
mechanism by which information transmission
persists across time in any real substrate.

Any associative structure with both properties
is a network.

No exceptions.

The parameters vary. The law does not.

## The Rhythmicity Threshold

A structure below the rhythmicity threshold —
where active information transmission has ceased —
is not a network. It is the residue of a former
network.

The no exceptions claim applies to networks
not to all matter.

The threshold is a precise mathematical object
derived in Paper 7.

## The Mathematical Form

f(t) = B + A·e^(-λt)·cos(ωt + φ)

Where:
- B = baseline associative coherence
- A = stimulus magnitude
- λ = recovery rate
- ω = characteristic carrier wave frequency
  of the network
- φ = phase offset

This function describes adaptive response in
any network at any scale.

Status: Well-supported hypothesis. Preliminary
empirical support at three independent scales.
Formal thermodynamic derivation pending Paper 11.

## The Measurement Decoupling Threshold

The observer effect is scale and complexity
dependent — not universally fixed as in
quantum mechanics.

Below the threshold — fragile systems,
systems in cascade precursor — instrument
coupling into the observed system is
non-negligible. Measurement disturbs.

Above the threshold — sufficiently complex
dissipative networks with sufficient coupling
coherence — instrument-system coupling becomes
negligible relative to system-internal coupling.
Non-entangled measurement is possible.

This is not an appropriation of quantum
mechanics. It is a distinct phenomenon with
structural parallels. Named precisely to avoid
confusion: the measurement decoupling threshold
not the Heisenberg boundary.

Derivation: Paper 11.
Application: Papers 8, 9, 12.
Near-threshold measurement: Paper 9,
extended section. Elevation to separate
paper flagged if mathematics requires it.

necessary and sufficient conditions derived
